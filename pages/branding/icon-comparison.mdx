## Hoe werkt dit?

Deze showcase draait op een **feature branch** (`feature/icon-showcase`).

### Om terug te gaan naar main (zonder icons):
```bash
cd ~/Workspace/kroescontrol-docs-public
git checkout main
```

### Om deze icons te behouden:
```bash
# Feature branch mergen naar main
git checkout main
git merge feature/icon-showcase
git push
```

### Om icons weer te verwijderen (na merge):
```bash
# Revert de merge commit
git revert -m 1 HEAD
git push
```
